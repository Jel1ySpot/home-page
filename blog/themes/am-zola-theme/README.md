# am-zola-theme

am-zola-theme 是一个受 Apple Music 启发的 Zola 主题，旨在提供现代、流畅且极具视觉吸引力的浏览体验。

## 主题特色

- **Apple Music 风格 UI**：提供直观的导航体验。
- **动态视觉效果**：内置基于 Shader 的动态网格渐变背景（Mesh Gradient），为页面增添活力。
- **高度可定制**：支持通过 `config.toml` 自定义主题色和背景渐变色板。
- **现代化设计**：注重细节的排版与交互动画。
